## TITLE: Product Pricing and Discount Management System

## DESCRIPTION

### A comprehensive system designed to efficiently manage product pricing, create and track discounts, and schedule promotional periods. It allows businesses to automate price adjustments, apply special offers, and ensure accurate and consistent pricing across all products. With this system, businesses can enhance sales strategies, streamline promotions, and provide customers with timely discounts and offers.

## PREREQUISITES

### Before running the system, make sure you have the following:
In Laptop, Pc or Android 
 - Visual Studio Code
 - Pydroid 3
 - Pycode
 - Google Chrome
	
## INSTALLATION

### Step 1: Download files or Use git clone from my github link

here's my github link
https://github.com/zaii-art/INTERMEDIATE-PROGRAMMING-PROJECT.git

### Step 2: In VsCode search Python in Extension after Install go to command prompt install flask using this command "pip install flask" 

### Step 2.1: In Android open pydroid or pycode then in terminal you will command using "pip install flask" 

### Step 3: After installing the Flask find the file name app.py then run

## Usage

- Open Chrome Browser/Google Chrome
- Go to http://127.0.0.1:5000
- View Features, Description, About us and tap Get started or Login
- Login as Admin or Staff
- Register an Account
